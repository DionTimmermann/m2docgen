reduceText 

 reduce txt to only to comments lines

Description: 

   text file is recuded to only its comment lines

Syntax: 

   obj.reduceText;

Input: 

   txt - file text: "string" || char
       Text of the file you want to process

Output: 

   redTxt - file text: "string"
       Text of reduced input file text.
